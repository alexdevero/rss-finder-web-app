# RSS Finder Web App

[Live App](https://rss-finder-web-app.vercel.app/)

Web app to help you find URLs for existing RSS feeds from website URLs.

### Table of contents

* [Install](#install)
* [Usage](#usage)
* [Support this project](#support-this-project)

* [License](#license)

### Install

#### Clone this repo

```bash
git clone https://github.com/alexdevero/rss-finder-web-app.git
```

#### Install dependencies

```bash
npm install
```

or

```bash
yarn
```

### Usage

#### Run the web app

```bash
npm run start
```

or

```bash
yarn run start
```

#### Build the web app

```bash
npm run build
```

or

```bash
yarn run build
```
